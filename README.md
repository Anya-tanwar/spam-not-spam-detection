# Spam Email Detector
https://spam-not-spam-detection.streamlit.app
## Overview
A Machine Learning-based Spam Email Detection web application that classifies email messages as Spam or Not Spam.

## Features
- Detects spam emails using NLP techniques
- Text preprocessing with NLTK
- Machine Learning classification model
- Interactive web interface using Streamlit
- Deployed on Streamlit Cloud

## Technologies Used
- Python
- Scikit-learn
- NLTK
- Pandas
- NumPy
- Streamlit

## Project Structure
